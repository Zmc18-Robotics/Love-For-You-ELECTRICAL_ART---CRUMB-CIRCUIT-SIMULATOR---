Here Material that i use : 

- Arduino nano x1
- LM555 x4
- Led pin x? ( ± 100 )
- Capacitor 33uF x2
- Capacitor 1uF x1
- Capacitor 100nF x1
- Resistor 220Ω x2
- Resistor 560Ω x17
- Resistor 100Ω x1
- Resistor 1000Ω x2
- Resistor 10000Ω x2
- Resistor 68Ω x2
- Resistor 1000000Ω x2
